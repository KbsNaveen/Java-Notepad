    Welcome to Java-Notepad Project Page.

    This is a small java project, that I created after the java course in 2nd year of my engineering, to develop an editor with all the basic features. Since the editor is developed in java it runs on any platform(Portable).

    Download and try it

    Download the project in either of the formats available(zip or tar ball) and extract the files to a folder, say Java-Notepad.

    Running

    You can run the project in two ways.

    • From the terminal( or console or command line or command prompt)
    • Using eclipse

    Running from console

    Open terminal or command prompt in the folder extracted(Java-Notepad). If you don't know how to do it, here it is:

    cd <Path to the folder(Java-Notepad)>

    Type at the prompt, replacing "" with actual path to the folder extracted.

    Now compile and run the project by typing the below commands:

    mkdir bin
javac -d bin src\*.java
java -cp bin StartJNotePad

    As soon as you type the last command you will see editor starting up.

    Running Using Eclipse

    Eclipse is one of the popular IDEs for java development. So you might already be using it for developing java codes. I added the required support files to open up the project in eclipse. All you need to do is to import the project into eclipse. If you don't know how to do it follow the below steps.

    * Fire up(or start) Eclipse.
* Click on File -> Import
* In the Import dialog box, type "Existing project" in the textbox
  (or) expand general and select "Existing Projects into Workspace".
* Click Next.
* Select the root directory by browsing to the folder extracted(Java-Notepad).
* Click Finish. Eclipse will load the project.

    Once the project is loaded into eclipse workspace, click on the run button to run the project. That's it you will see the editor starting up. (Select "Java Application" if you see any dialog box asking the "Run as" type of the applicaion.)
